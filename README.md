# Do One Thing Today

**One tiny task. One big streak. Focus on what matters today.**  

A simple, minimal PWA built with Vue 3 and Vite.  
No accounts. No clutter. Just one daily task to help you stay focused.  

## Features

- Add **one task** for today
- Mark the task as **done**
- Track your **streak** (days in a row you completed your task)
- **LocalStorage only** â€” everything stays on your device
- Installable on mobile or desktop, just like an app

```bash
# Clone the repo
git clone https://github.com/YOUR_USERNAME/do-one-thing-today.git

# Install dependencies
